Home Page, At Rest and Opened

The concept Eric named: class selection is a menu that OPENS on the home page before queueing. State A is what a brand-new player sees in their first five seconds — the game already breathing behind the menu, a callsign, a pre-picked class, and one dominant amber PLAY (a first-timer never has to open anything). State B is the same page with the class layer open: four launch classes as cards side by side, built to scale past four. Silhouettes and Regatta hexes lifted from the ratified Class Identity Board. Player's preference pick in this mock: Cyan.

Layout is a proposal for reaction · pip values are placeholder
STATE A

Home page at rest — five seconds to cool

The CIC picture (rings, sweep, live blips) idles behind the menu so the page IS the game, not a lobby about the game. Reading order top-to-bottom: wordmark → callsign → current-class chip (compact — one glance says what you'll sail, click to open State B) → color preference strip → PLAY, the single dominant action. Gunboat is the proposed first-run default ("speedy boy with some guns" — the friendliest fantasy); returning players keep their last pick. "How to Play" is the GDD E7 entry point, deliberately quiet.

STATE B

Class layer open — the Hades weapon pick, cards side by side

Same page; the chip (or PLAY-adjacent "change class") opened this layer over a dimmed home. Each card carries the four differences that matter: hull silhouette (identity-board geometry), the GDD power-fantasy line verbatim, envelope feel as three pip scales (SPEED / TOUGHNESS / TURNING — placeholder values, flagged on every card), and the fitted loadout (universal gun + the two class specials from the GDD loadout table). The selected class wears the player's Regatta color (bright outline, darker fill); unselected hulls stay silver linework. Scale-past-4: the row is a horizontal scroll rail — a dashed ghost card ("MORE CLASSES IN DEVELOPMENT") sits fifth and is deliberately clipped by the frame edge, under a right-edge fade and a phosphor chevron, so the eye learns the shelf keeps going.
