Forge Report — The Identity Fork

Brainstorm idea #86 · lobby-picked identity vs match-built identity · Hullcracker.io · forged with Eric, 2026-07-15
Verdict: HARDENED. The idea arrived as a binary — "Apex-style lobby kits vs roguelike match-built builds" — and left as an architecture: a lobby promise that never misses, plus match-built growth that RNG is allowed to govern. The binary itself was the flaw; both reference games (Hades, Apex) had already split the difference the same way. Ready to feed a Game Brief / GDD.

Locked Decisions

  • Promise + growth, not promise vs growth. Lobby pick = a genuinely different loadout live at 0:00; the upgrade deck grows it in-match.
  • Classes differ by loadout, not size. Verb composition from a shared armory — focus, not exclusivity. Slot taxonomy deliberately unpinned.
  • The Hades contract: the promise never misses; RNG only touches what was never promised. That line is what separates "lame roll, one more run" from "the menu lied."
  • Class is an envelope; a build is a point inside it. Two endgame Torpedo Boats: guessable threat range, never identical.
  • Anti-snowball outranks the Conservation Law. Hiding is a viable strategy by design; the massacre-player must stay challengeable. The passive tick (~1 pick/min, zone closed at 10:00) is the anti-snowball floor — the brainstorm's "session thesis" (#42) is hereby demoted to a tendency.
  • 20–30 humans is the lobby target (6 is far too low); fill-or-timer, never hostage; custom lobbies adopted; bot-fill disfavored.
  • Noise law: information noise must never obscure the player-vs-player hunt. PvE clusters: small, capped, maybe map-marked.
  • Radar grows shape. Blip size/shape carries class legibility beyond truesight; radar is a hunting tool.

Killed & Rejected — Cause of Death

  • The fork as a binary — both poles dead on contact with the reference games it cited.
  • "Everyone has gun/torpedo/mine" — the universal thing is the slot grammar, not the contents.
  • PvE fleets as mandatory density; bot-fill in player lobbies; ambiguity-as-thrill — killed by the noise law and by Eric declining ownership of the doc's #65 framing.

Weak Points That Survived Scrutiny

  • The Unwitnessed Build (accepted openly): ten picks must become felt — seen, heard, feared — or match-built identity is a spreadsheet. Unsolved.
  • The kill-bonus ratio is the single number pricing the rat. Untuned.
  • Ghost ocean: map scales by lobby cap, not actual starting roster (verified in code) — fix by sizing from the roster at countdown end.
  • Open by choice: eclipse dial, deck-merge mechanism, slot taxonomy, class list (the Carrier is the stress test), covenant scaling. All prototype/playtest questions — cheapest experiment: one hand-built Cruiser loadout variant + weighted offers.
