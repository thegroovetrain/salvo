# Forged: The Identity Fork (#86) — HARDENED

**The fork is dissolved, not won.** Player identity = **lobby promise + match-built growth**. Not Apex vs roguelike — both, with a contract between them.

## Locked

1. **The promise**: the lobby pick puts a genuinely different loadout in your hands at 0:00. If classes exist they MUST differ — verb composition from a shared armory, **focus not exclusivity** (Torpedo Boat is torp-focused, not torp-exclusive; a Battleship might swap torps for a heavier secondary battery). Slot taxonomy (basic/special/class, maybe a 4th intel/evasion slot) deliberately unpinned.
2. **The contract** (Hades model): the promise never misses; RNG only governs what was never promised. A failed build roll is "one more run" fuel only because the menu never lied.
3. **The envelope**: class = envelope, build = point inside it. Two endgame same-class ships: guessable threat range, non-identical builds.
4. **Anti-snowball outranks the Conservation Law.** Hiding/playing safe is a *viable strategy by design* (healing scarcity already prices recklessness); the massacre-player must never become unchallengeable. Passive XP tick (~1 upgrade/min, zone closed at 10:00) is the anti-snowball floor, not a leak. Kill bonuses stay modest. **This demotes the brainstorm doc's "session thesis" (#42) to a tendency — the Game Brief inherits THIS version.**
5. **Lobby size**: 20–30 humans target (maybe 20 solo / 40 duos / 60 trios). 6 is far too low. Never hold lobbies hostage: min 2 teams queued, fill-or-timer, start. Custom/private lobbies adopted. Bot-fill in standard lobbies disfavored — launch small if small.
6. **Noise principle**: information noise must never obscure the player-vs-player hunt. PvE drone clusters allowed only as small, capped, possibly map-marked XP pockets.
7. **Radar fidelity direction**: returns gain shape/size (real-radar flavored, fun-biased) → class legibility at blip range (a battleship paints bigger). Radar is a hunting tool.

## Rejected

- **Pure lobby-kits and pure match-built identity** — both poles dead; reference games (Hades, Apex) both split promise/growth.
- **"Everyone has gun/torpedo/mine"** — wrong frame; universal is the *slot grammar*, contents may vary per class.
- **PvE fleets as mandatory world density**; **ambiguity-as-thrill** (doc's #65 framing — Eric never owned it); **bot-fill in player lobbies**.

## Open on purpose (prototype/playtest, not talk)

- **The Unwitnessed Build** — biggest accepted weak point: 10 picks must become *felt* (audio, hull visuals, on-water behavior), or match-built identity is a spreadsheet. Unanswered.
- Eclipse dial (off-class builds priced by deck weighting — proposal, Eric ambivalent), deck-merge mechanism (generic + class deck), slot taxonomy, the class list (Carrier #92 is the stress test).
- Kill-bonus ratio (the one number pricing the rat), covenant scaling (10 min/20p → maybe 20 min/60p; lean "10 is fine").
- Ghost-ocean fix: map scales by playerCap, not actual starting roster (verified constants.ts:11) — size from roster at countdown end.
- **Cheapest experiment**: hand-build ONE loadout variant of the Cruiser + weighted offers; if first contact feels different and endgame builds diverge, the architecture is validated.
